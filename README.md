# Business Card Project

Welcome to my first coding project! This repository contains my personal business card website, built using HTML and CSS.

## 🚀 About This Project

This project marks the beginning of my journey into software development. As a beginner, I wanted to create a tangible project to apply the fundamentals I've been learning. While the process has had its challenges, I have truly enjoyed the problem-solving aspect of coding.

## 🛠️ Technologies Used

- **HTML5:** For the structural framework of the page.
- **CSS3:** For styling, layout (Flexbox), and design.

## 💡 Key Learnings & Analogies

One of the biggest takeaways from this project is understanding the relationship between HTML and CSS. I like to think of it like building a house:
- **HTML** acts as the **structure or framework** of the house (the walls, the rooms).
- **CSS** is the **renovation and interior design**. It decides where the furniture goes in each room (labeled element) and how it looks.

## ⚠️ Challenges Faced

Learning to code is a process of trial and error. Here are some specific hurdles I overcame during this challenge:

1.  **File Paths:** Initially, I struggled with correctly linking my CSS file to my HTML file. Learning how relative paths work was a crucial first step.
2.  **Responsive Layouts (Flexbox):** I faced significant issues with margins and Flexbox. Specifically, when I used fixed `height` and `width` properties, the layout would break or shift unexpectedly whenever I resized the browser window.
    - *Solution:* I managed to resolve the immediate layout issues by adjusting the dimensions, though I recognize that using fixed units isn't the most effective method for responsiveness. This has motivated me to learn more about relative units (like `%` or `rem`) and media queries for my next project.
